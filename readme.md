# TaildwindCss Coffee Style by Duc
## Requirements
* **nodejs >= v14.7.0**
* **npm >= v6.14.7**

Clone project and run test.
```

$ git clone https://github.com/Tienduc37y/coffee-tailwindcss.git
$ cd <project_dir>
$ npm run build:css

```
